> A full-stack event booking web application built with Laravel 13 and MySQL. Users can browse events, book seats, and manage reservations through a polished, responsive interface.

---

## Features

- **Event Discovery** ‚Äî browse and filter events by location, date, and availability
- **Seat Booking** ‚Äî real-time seat tracking with progress indicators
- **Secure Checkout** ‚Äî simulated payment flow with credit/debit card and bank transfer
- **User Dashboard** ‚Äî view, manage, and cancel bookings
- **Admin Panel** ‚Äî create/edit/cancel events, view all bookings and payments
- **Role-based Access** ‚Äî user and admin roles with middleware protection
- **Responsive Design** ‚Äî mobile-first UI with custom Bootstrap 5 theming

## Quick Start (Local)

**Prerequisites:** PHP 8.3+, Composer, MySQL 8.0+, Node.js

```bash
# 1. Clone the repo
git clone https://github.com/abdullahkousa2/eventsy.git
cd eventsy

# 2. Install dependencies
composer install
npm install && npm run build

# 3. Configure environment
cp .env.example .env
php artisan key:generate

# 4. Set your database credentials in .env
#    DB_DATABASE=event_booking
#    DB_USERNAME=root
#    DB_PASSWORD=yourpassword

# 5. Run migrations and seed demo data
php artisan migrate --seed

# 6. Start the dev server
php artisan serve
```

Visit `http://localhost:8000`

---

## Docker Setup

```bash
# Build and start all services (app + MySQL)
docker compose up --build

# First run applies migrations and seeds automatically
# Visit http://localhost:8000
```

To stop:
```bash
docker compose down          # keep database volume
docker compose down -v       # remove database volume too
```

## Environment Variables

Key variables to configure in `.env`:

| Variable | Description |
|---|---|
| `APP_KEY` | Generated by `php artisan key:generate` |
| `DB_HOST` | MySQL host (`127.0.0.1` local, `db` in Docker) |
| `DB_DATABASE` | Database name (`event_booking`) |
| `DB_USERNAME` | MySQL user |
| `DB_PASSWORD` | MySQL password |
| `APP_DEBUG` | Set to `false` in production |
